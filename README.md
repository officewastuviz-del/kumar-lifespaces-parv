# Kumar Lifespaces Parv — WIN Experience

An immersive, customer-facing website for Kumar Lifespaces Parv at Moshi,
Pune. The experience uses chaptered project films for the arrival, amenities,
wellness spaces and signature architecture.

## Live website

https://officewastuviz-del.github.io/kumar-lifespaces-parv/

## Experience

- Cinematic, full-screen project film
- 2 & 3 BHK residence overview
- Amenity-specific video chapters
- Actual Kumar Parv Google Maps location
- No lead forms, analytics, tracking, dashboards or AR

The public GitHub Pages release is assembled by
`.github/workflows/pages.yml`. The Vinext source used for the private Sites
release remains in `app/`.
